# 📌 17.4 — Polyfills Mastery (The Definitive Guide)

## ① INTRODUCTION TO POLYFILLS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A **polyfill** is a piece of code (usually JavaScript on the web) used to provide modern functionality on older browsers that do not natively support it. 

In a technical interview, implementing a polyfill is not just about writing code that works; it is a test of your deep understanding of:
1. **The ECMAScript specification** (how standard built-ins are specified to behave in edge cases).
2. **Context Binding & `this` mechanics** (how scopes are resolved and bound dynamically).
3. **Memory management & optimization** (avoiding leaks, memory bloat, and JIT deoptimizations in engines like V8).
4. **Prototypes and the inheritance chain** (attaching functions to native prototypes safely).

## ② THE HOLY TRINITY: `call`, `apply`, AND `bind`
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Understanding how to build `call`, `apply`, and `bind` from scratch is the most common prerequisite for mastering advanced JavaScript. All three methods belong to `Function.prototype` and are used to manipulate the execution context (`this`) of functions.

### 🔹 Conceptual Deep Dive: Context Manipulation
In JavaScript, `this` is evaluated dynamically when a function is invoked (with the exception of arrow functions, which resolve `this` lexically). By default, a function's `this` refers to the object it was called on (the receiver). 

The core strategy of rewriting these methods is **temporary object binding**:
1. Attach the function as a property of the target object context.
2. Invoke that property (which dynamically sets its `this` to that object).
3. Delete the temporary property to avoid modifying/polluting the target object.

---

### 1️⃣ `Function.prototype.call` Polyfill

`call()` invokes a function with a given `this` value and arguments provided individually.

#### The V8 Engine Edge Cases & Safety Protocols:
- **Global Fallback:** If the `thisArg` is `null` or `undefined`, it must fall back to `globalThis` (e.g. `window` in browsers, `global` in Node.js).
- **Primitive Autoboxing:** If `thisArg` is passed as a primitive value (like a `number`, `string`, or `boolean`), JavaScript natively wraps it in its corresponding object prototype (e.g., `5` becomes `Number(5)` object). We use `Object(thisArg)` to mirror this behavior.
- **Property Collision Prevention:** If we assign our function to a hardcoded property key on the target object (like `thisArg.fn = this`), we risk overwriting an existing property `fn`. To prevent this, we use a unique ES6 `Symbol` as the key.

#### Flowchart of `myCall` execution:
```mermaid
graph TD
    A[Function.prototype.myCall] --> B{Is thisArg null/undefined?}
    B -- Yes --> C[thisArg = globalThis]
    B -- No --> D{Is thisArg a primitive?}
    D -- Yes --> E[Autobox: Object(thisArg)]
    D -- No --> F[Use thisArg directly]
    C --> G[Generate Unique key: Symbol]
    E --> G
    F --> G
    G --> H[Assign function context 'this' to thisArg[Symbol]]
    H --> I[Invoke function with spread arguments: thisArg[Symbol]\(...args\)]
    I --> J[Capture return value]
    J --> K[Delete temporary Symbol property]
    K --> L[Return captured value]
```

#### Code Implementation:
```javascript
/**
 * Polyfill for Function.prototype.call
 * @param {Object|any} thisArg - The context to bind to the function
 * @param {...any} args - List of arguments to pass to the function
 * @returns {any} Result of the function invocation
 */
Function.prototype.myCall = function(thisArg, ...args) {
    // 1. Ensure that 'this' is a callable function
    if (typeof this !== 'function') {
        throw new TypeError(this + ' is not a function');
    }

    // 2. Handle null or undefined contexts
    if (thisArg === null || thisArg === undefined) {
        thisArg = globalThis;
    } else {
        // 3. Autobox primitives (numbers, strings, booleans) into objects
        thisArg = Object(thisArg);
    }

    // 4. Generate a unique property key to prevent key collision
    const fnKey = Symbol('fn');

    // 5. Temporarily attach the function (represented by 'this') to the context object
    thisArg[fnKey] = this;

    // 6. Invoke the function using the method call pattern so its 'this' points to 'thisArg'
    const result = thisArg[fnKey](...args);

    // 7. Delete the temporary property to keep the object clean
    delete thisArg[fnKey];

    // 8. Return the output of the function execution
    return result;
};
```

### 2️⃣ `Function.prototype.apply` Polyfill

`apply()` is identical to `call()`, except that arguments are passed as an array (or array-like object).

#### Spefication Details:
- If the second argument is neither `null` nor `undefined` and is not an object (e.g., passing a primitive like `5`), a `TypeError` must be thrown.
- We must convert the arguments parameter into a real array using `Array.from()` if it is array-like, or fall back to an empty array `[]` if it is nullish.

#### Code Implementation:
```javascript
/**
 * Polyfill for Function.prototype.apply
 * @param {Object|any} thisArg - The context to bind to the function
 * @param {Array|any} argsArray - Array-like object containing arguments
 * @returns {any} Result of the function invocation
 */
Function.prototype.myApply = function(thisArg, argsArray) {
    // 1. Ensure 'this' is a function
    if (typeof this !== 'function') {
        throw new TypeError(this + ' is not a function');
    }

    // 2. Validate arguments list
    if (argsArray !== null && argsArray !== undefined && typeof argsArray !== 'object') {
        throw new TypeError('CreateListFromArrayLike called on non-object');
    }

    // 3. Handle null/undefined contexts and autobox primitives
    if (thisArg === null || thisArg === undefined) {
        thisArg = globalThis;
    } else {
        thisArg = Object(thisArg);
    }

    const fnKey = Symbol('fn');
    thisArg[fnKey] = this;

    // 4. Safely convert array-like parameter to true array or default to empty list
    const args = argsArray ? Array.from(argsArray) : [];

    const result = thisArg[fnKey](...args);
    delete thisArg[fnKey];
    return result;
};
```

### 3️⃣ `Function.prototype.bind` Polyfill (The Ultimate Interview Challenge)

`bind()` returns a **new function** that, when called, has its `this` keyword set to the provided value, with a given sequence of arguments preceding any provided when the new function is called (currying).

#### ⚠️ The Constructor Trap (How to Fail an Interview):
Most naive implementations of `bind` return a wrapper like this:
```javascript
// NAIVE AND INCOMPLETE
Function.prototype.badBind = function(thisArg, ...boundArgs) {
    const fn = this;
    return function(...newArgs) {
        return fn.apply(thisArg, [...boundArgs, ...newArgs]);
    };
};
```
**Why this fails:** If the returned bound function is invoked as a constructor using the `new` keyword (e.g., `const instance = new BoundFunction()`), the `thisArg` parameter **must be ignored**. The `this` context inside the function must instead be the newly created object, inheriting the original function's prototype.

#### How to Solve the Constructor Trap:
1. Detect if the function is called as a constructor:
   - In ES6, we can check `new.target` inside the wrapper function.
   - For complete backwards compatibility, we check `this instanceof boundFn`. If the function is instantiated with `new`, the execution context's `this` will be an instance of the returned wrapper function.
2. Link prototypes: The returned bound function must share the prototype chain of the target function. We use `Object.create(originalFunction.prototype)` to link them safely without sharing the exact same reference.

#### Flowchart of `myBind` constructor resolution:
```mermaid
graph TD
    A[Function.prototype.myBind] --> B[Return Wrapper Function: boundFn]
    B --> C{Called with 'new' keyword?}
    C -- Yes (Constructor) --> D[this instanceof boundFn is true]
    D --> E[Use 'this' as context (newly created object instance)]
    C -- No (Regular Call) --> F[Use passed 'thisArg' as context]
    E --> G[Merge boundArgs + newArgs]
    F --> G
    G --> H[Invoke original function: originalFn.apply(context, mergedArgs)]
    H --> I[Maintain prototype: boundFn.prototype = Object.create(originalFn.prototype)]
```

#### Code Implementation:
```javascript
/**
 * Polyfill for Function.prototype.bind
 * @param {Object|any} thisArg - The context to bind
 * @param {...any} boundArgs - Arguments prepended to the final call
 * @returns {Function} A new bound function wrapper
 */
Function.prototype.myBind = function(thisArg, ...boundArgs) {
    // 1. Ensure 'this' is a function
    if (typeof this !== 'function') {
        throw new TypeError(this + ' must be bound');
    }

    const originalFn = this;

    // 2. Return the bound function wrapper
    function boundFn(...newArgs) {
        // 3. Resolve execution context:
        // If 'boundFn' is called as a constructor (using 'new'), 'this' is a new object
        // created with boundFn's prototype. If so, override 'thisArg' and use 'this'.
        const isConstructor = this instanceof boundFn;
        const context = isConstructor ? this : thisArg;

        // 4. Merge pre-bound arguments and invocation arguments, then run the function
        return originalFn.apply(context, [...boundArgs, ...newArgs]);
    }

    // 5. Connect the prototype chains to preserve the 'instanceof' operator
    if (originalFn.prototype) {
        // Use Object.create to create a prototype link without mutating originalFn's prototype
        boundFn.prototype = Object.create(originalFn.prototype);
    }

    return boundFn;
};
```

## ③ ARRAY PROTOTYPE POLYFILLS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Array prototype methods must handle **sparse arrays** (arrays with missing/empty slots, e.g. `[1, , 3]`). According to the spec, most array methods skip indices that are not set in the array's backing store map. We check this using the `index in array` operator.

### 1️⃣ `Array.prototype.map`
Creates a new array populated with the results of calling a provided function on every element in the calling array.

```javascript
Array.prototype.myMap = function(callback, thisArg) {
    if (this == null) {
        throw new TypeError('Array.prototype.myMap called on null or undefined');
    }
    if (typeof callback !== 'function') {
        throw new TypeError(callback + ' is not a function');
    }

    const O = Object(this);
    const len = O.length >>> 0; // Cast length to uint32 (converts NaN to 0, float to int)
    const A = new Array(len);

    for (let k = 0; k < len; k++) {
        // 'k in O' ensures sparse slots (holes) are skipped and not evaluated
        if (k in O) {
            A[k] = callback.call(thisArg, O[k], k, O);
        }
    }
    return A;
};
```

### 2️⃣ `Array.prototype.filter`
Creates a shallow copy of a portion of a given array, filtered down to just the elements from the given array that pass the test implemented by the provided function.

```javascript
Array.prototype.myFilter = function(callback, thisArg) {
    if (this == null) {
        throw new TypeError('Array.prototype.myFilter called on null or undefined');
    }
    if (typeof callback !== 'function') {
        throw new TypeError(callback + ' is not a function');
    }

    const O = Object(this);
    const len = O.length >>> 0;
    const res = [];

    for (let k = 0; k < len; k++) {
        if (k in O) {
            if (callback.call(thisArg, O[k], k, O)) {
                res.push(O[k]);
            }
        }
    }
    return res;
};
```

### 3️⃣ `Array.prototype.reduce`
Executes a user-supplied "reducer" callback function on each element of the array, in order, passing in the return value from the calculation on the preceding element.

```javascript
Array.prototype.myReduce = function(callback, initialValue) {
    if (this == null) {
        throw new TypeError('Array.prototype.myReduce called on null or undefined');
    }
    if (typeof callback !== 'function') {
        throw new TypeError(callback + ' is not a function');
    }

    const O = Object(this);
    const len = O.length >>> 0;
    let k = 0;
    let accumulator;

    // Check if initialValue is provided (note: we check arguments.length, not initialValue !== undefined)
    if (arguments.length >= 2) {
        accumulator = initialValue;
    } else {
        // If initialValue isn't passed, find the first non-empty index in the array to act as accumulator
        let kPresent = false;
        while (k < len && !kPresent) {
            kPresent = k in O;
            if (kPresent) {
                accumulator = O[k];
            }
            k++;
        }
        if (!kPresent) {
            throw new TypeError('Reduce of empty array with no initial value');
        }
    }

    for (; k < len; k++) {
        if (k in O) {
            accumulator = callback(accumulator, O[k], k, O);
        }
    }
    return accumulator;
};
```

### 4️⃣ `Array.prototype.forEach`

### 5️⃣ `Array.prototype.flat`
Creates a new array with all sub-array elements concatenated into it recursively up to the specified depth.

```javascript
Array.prototype.myFlat = function(depth = 1) {
    if (this == null) {
        throw new TypeError('Array.prototype.myFlat called on null or undefined');
    }

    const O = Object(this);
    const len = O.length >>> 0;
    const d = depth === undefined ? 1 : Number(depth);

    function flatten(arr, currentDepth) {
        if (currentDepth < 1) {
            // Return shallow copy skipping holes
            const result = [];
            for (let i = 0; i < arr.length; i++) {
                if (i in arr) result.push(arr[i]);
            }
            return result;
        }

        const result = [];
        for (let i = 0; i < arr.length; i++) {
            if (i in arr) {
                const val = arr[i];
                if (Array.isArray(val)) {
                    result.push(...flatten(val, currentDepth - 1));
                } else {
                    result.push(val);
                }
            }
        }
        return result;
    }

    return flatten(O, d);
};
```

## ④ ASYNCHRONOUS PROMISE COMBINATORS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Promise combinators orchestrate multiple promises. Understanding how to polyfill them is a core test of asynchronous flow control.

### 1️⃣ `Promise.all`
Resolves only when **all** input promises resolve. Rejects immediately if **any** input promise rejects.

```javascript
Promise.myAll = function(iterable) {
    return new Promise((resolve, reject) => {
        try {
            const promises = Array.from(iterable);
            const results = [];
            let completed = 0;

            if (promises.length === 0) {
                resolve(results);
                return;
            }

            promises.forEach((promise, index) => {
                // Wrap in Promise.resolve to handle non-promise raw values correctly
                Promise.resolve(promise)
                    .then(val => {
                        results[index] = val;
                        completed++;
                        // Resolve only when all elements are loaded
                        if (completed === promises.length) {
                            resolve(results);
                        }
                    })
                    .catch(reject); // Propagate rejection immediately
            });
        } catch (error) {
            reject(error); // Capture error in case iteration itself throws
        }
    });
};
```

### 2️⃣ `Promise.allSettled`
Resolves when **all** input promises have settled (either resolved or rejected). It returns an array of objects detailing the outcome of each promise.

```javascript
Promise.myAllSettled = function(iterable) {
    return new Promise((resolve, reject) => {
        try {
            const promises = Array.from(iterable);
            const results = [];
            let settledCount = 0;

            if (promises.length === 0) {
                resolve(results);
                return;
            }

            promises.forEach((promise, index) => {
                Promise.resolve(promise)
                    .then(val => {
                        results[index] = { status: 'fulfilled', value: val };
                    })
                    .catch(err => {
                        results[index] = { status: 'rejected', reason: err };
                    })
                    .finally(() => {
                        settledCount++;
                        if (settledCount === promises.length) {
                            resolve(results);
                        }
                    });
            });
        } catch (error) {
            reject(error);
        }
    });
};
```

### 3️⃣ `Promise.any`
Resolves as soon as **any** of the input promises resolve. Rejects only when **all** promises reject (returning an `AggregateError`).

```javascript
Promise.myAny = function(iterable) {
    return new Promise((resolve, reject) => {
        try {
            const promises = Array.from(iterable);
            const errors = [];
            let rejectedCount = 0;

            if (promises.length === 0) {
                reject(new AggregateError([], 'All promises were rejected'));
                return;
            }

            promises.forEach((promise, index) => {
                Promise.resolve(promise)
                    .then(resolve) // Resolves immediately with the first fulfillment
                    .catch(err => {
                        errors[index] = err;
                        rejectedCount++;
                        if (rejectedCount === promises.length) {
                            reject(new AggregateError(errors, 'All promises were rejected'));
                        }
                    });
            });
        } catch (error) {
            reject(error);
        }
    });
};
```

### 4️⃣ `Promise.race`
Settles as soon as **any** of the input promises settles (either resolves or rejects).

```javascript
Promise.myRace = function(iterable) {
    return new Promise((resolve, reject) => {
        try {
            const promises = Array.from(iterable);
            // An empty iterable remains permanently pending (matching native behavior)
            promises.forEach(promise => {
                Promise.resolve(promise).then(resolve, reject);
            });
        } catch (error) {
            reject(error);
        }
    });
};
```

### 1️⃣ `Object.create`
Creates a new object, using an existing object as the prototype of the newly created object.

```javascript
Object.myCreate = function(proto, propertiesObject) {
    // 1. Prototype must be an object or null
    if (typeof proto !== 'object' && typeof proto !== 'function' && proto !== null) {
        throw new TypeError('Object prototype may only be an Object or null');
    }

    // 2. Establish a temporary constructor
    function F() {}
    F.prototype = proto;

    // 3. Create the new object inheriting prototype
    const obj = new F();

    // 4. Handle null prototype specifically
    if (proto === null) {
        Object.setPrototypeOf(obj, null);
    }

    // 5. Define descriptors if specified
    if (propertiesObject !== undefined) {
        if (propertiesObject === null) {
            throw new TypeError('Cannot convert first argument to object');
        }
        Object.defineProperties(obj, propertiesObject);
    }

    return obj;
};
```

### 2️⃣ `Object.assign`
Copies all enumerable own properties from one or more source objects to a target object. It returns the modified target object.

```javascript
Object.myAssign = function(target, ...sources) {
    if (target === null || target === undefined) {
        throw new TypeError('Cannot convert undefined or null to object');
    }

    const to = Object(target);

    for (let i = 0; i < sources.length; i++) {
        const nextSource = sources[i];

        if (nextSource !== null && nextSource !== undefined) {
            // 1. Assign all enumerable string properties
            const keys = Object.keys(nextSource);
            for (let j = 0; j < keys.length; j++) {
                const key = keys[j];
                to[key] = nextSource[key];
            }

            // 2. Assign all enumerable symbol properties
            const symbols = Object.getOwnPropertySymbols(nextSource);
            for (let j = 0; j < symbols.length; j++) {
                const sym = symbols[j];
                if (Object.prototype.propertyIsEnumerable.call(nextSource, sym)) {
                    to[sym] = nextSource[sym];
                }
            }
        }
    }
    return to;
};
```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 1️⃣ Deep Clone (V8 Spec Compliant)
Clones nested values, handling circular references, special object types (Date, RegExp, Map, Set), and maintaining symbols.

```javascript
/**
 * Deep clones an object resolving cyclic references
 * @param {any} value - Object to copy
 * @param {WeakMap} hash - Cache to store reference checks
 * @returns {any} Copied output
 */
function deepClone(value, hash = new WeakMap()) {
    // 1. Primitive types and null
    if (value === null || typeof value !== 'object') {
        return value;
    }

    // 2. Dates
    if (value instanceof Date) {
        return new Date(value.getTime());
    }

    // 3. RegExps
    if (value instanceof RegExp) {
        return new RegExp(value.source, value.flags);
    }

    // 4. Circular Reference lookup
    if (hash.has(value)) {
        return hash.get(value);
    }

    // 5. Maps
    if (value instanceof Map) {
        const newMap = new Map();
        hash.set(value, newMap);
        value.forEach((val, key) => {
            newMap.set(deepClone(key, hash), deepClone(val, hash));
        });
        return newMap;
    }

    // 6. Sets
    if (value instanceof Set) {
        const newSet = new Set();
        hash.set(value, newSet);
        value.forEach(val => {
            newSet.add(deepClone(val, hash));
        });
        return newSet;
    }

    // 7. Arrays or Objects (inheriting correct prototype chain)
    const cloneTarget = Array.isArray(value) ? [] : Object.create(Object.getPrototypeOf(value));
    hash.set(value, cloneTarget);

    // 8. Copy all descriptors (keys, symbols, and values)
    const keys = Reflect.ownKeys(value);
    for (const key of keys) {
        const desc = Object.getOwnPropertyDescriptor(value, key);
        if (desc) {
            desc.value = deepClone(value[key], hash);
            Object.defineProperty(cloneTarget, key, desc);
        }
    }

    return cloneTarget;
}
```

### 2️⃣ Debounce (With Immediate Trigger)
Limits the rate at which a function can fire, executing after `wait` milliseconds since the last call.

```javascript
function debounce(func, wait, immediate = false) {
    let timeoutId = null;

    return function(...args) {
        const context = this;

        const later = function() {
            timeoutId = null;
            if (!immediate) {
                func.apply(context, args);
            }
        };

        const callNow = immediate && !timeoutId;

        // Clear existing queue timer
        if (timeoutId) {
            clearTimeout(timeoutId);
        }

        // Set new trigger timeout
        timeoutId = setTimeout(later, wait);

        if (callNow) {
            func.apply(context, args);
        }
    };
}
```

### 3️⃣ Throttle (Trailing & Leading Execution)
Enforces a maximum number of times a function can be called over time.

```javascript
function throttle(func, wait) {
    let timeoutId = null;
    let lastRan = 0;

    return function(...args) {
        const context = this;
        const now = Date.now();
        const remaining = wait - (now - lastRan);

        if (remaining <= 0 || remaining > wait) {
            if (timeoutId) {
                clearTimeout(timeoutId);
                timeoutId = null;
            }
            lastRan = now;
            func.apply(context, args);
        } else if (!timeoutId) {
            timeoutId = setTimeout(function() {
                lastRan = Date.now();
                timeoutId = null;
                func.apply(context, args);
            }, remaining);
        }
    };
}
```
